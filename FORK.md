# Fork Notes

This repository is a personal/local fork of [cmagnussen/blitztext-app](https://github.com/cmagnussen/blitztext-app).

It exists to keep a working macOS build with local usability changes. It is not the official upstream repository, not a hosted service, and not a separate commercial product.

## What Is Retained

- The upstream MIT license and attribution.
- The native macOS app source code.
- Setup, privacy, local model, security, and support notes that are still useful for building and reviewing the app.
- Screenshots as illustrative documentation.

## What Is Changed

- Local macOS usability changes are documented in [CHANGELOG.md](CHANGELOG.md).
- Build instructions point to this fork.
- Support, contribution, and security notes are written for a personal fork instead of an official product.
- The README clearly states the upstream relationship.

## What Is Intentionally Not Included

- OpenAI API keys or other secrets.
- Local WhisperKit/CoreML model downloads.
- Built `.app` bundles.
- Xcode DerivedData or other build output.
- Local user settings, logs, recordings, or transcripts.
- A fork-specific product roadmap.
- Website, launch, or release-planning documents from the upstream project.

## Upstream Changes

If the upstream project changes, this fork can be updated by merging or cherry-picking upstream commits. Fork-specific changes should stay small and documented so future syncs remain manageable.
